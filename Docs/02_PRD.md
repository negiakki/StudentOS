# StudentOS Product Requirements Document (PRD)

**Version:** 1.0 (MVP)

**Status:** Feature Frozen

**Project:** StudentOS

**AI Assistant:** Coco

---

# 1. Overview

StudentOS is an AI-powered web application that acts as a daily operating system for college students.

Instead of switching between multiple applications for attendance, timetable, assignments, reminders and planning, students interact with one dashboard and one AI companion named Coco.

The objective of Version 1 is to provide students with a personalized daily dashboard that answers one question:

> **"What do I need to know today?"**

---

# 2. MVP Objectives

The MVP must:

* Reduce the number of apps students use daily.
* Minimize manual data entry.
* Make attendance management effortless.
* Automatically generate a timetable from uploaded PDFs/images.
* Provide an AI assistant with access to student data.
* Create a habit where students open StudentOS every morning.

# 20. Acceptance Criteria

StudentOS Version 1 is complete when:

A new student can create an account.

Upload a timetable.

Confirm parsed classes.

Enter attendance.

Open the dashboard.

Receive a Daily Brief.

Ask Coco questions about their academic schedule.

Update attendance in under 30 seconds.

Return the next day and continue using the system without repeating onboarding.
